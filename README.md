# 💕 Anniversary Tribute for Mom & Dad 🌹

A beautiful, interactive anniversary celebration webpage created with ❤️ by Surekha.

## ✨ Features

🌹 **Floating Hearts & Rose Petals** - Continuous romantic animations
📸 **Photo Slideshow** - Display your favorite family memories
🎵 **Background Music** - Optional soft music player
💌 **Heartfelt Message** - Your personalized anniversary message
✨ **Sparkle Effects** - Special celebration animations
🎉 **"Celebrate Now!" Animation** - Interactive celebration button with floating effects

## 🚀 Live Demo

Your anniversary webpage is ready to go live!

### To Enable GitHub Pages:
1. Go to repository **Settings** → **Pages**
2. Select **Main** branch as source
3. Your site will be live at: `https://surekhaprakash14.github.io/parents-anniversary/`

Works beautifully on:
- 💻 Desktop browsers

## 🎯 Quick Start

1. **Update Photos**: Replace placeholder photos with your memories
2. **Edit Message**: Customize the anniversary message
3. **Choose Music**: Select background music (optional)
4. **Enable Pages**: Activate GitHub Pages to make it live
5. **Share**: Send the link to your parents!

## 🎭 Interactive Elements

- **⬅️ Previous / Next ➡️** - Navigate through photos
- **● Dots** - Click to jump to specific photo
- **🎵 Music Player** - Pause/play background music
- **✨ Celebrate Now!** - Trigger special celebration effects
- **⌨️ Keyboard Navigation** - Use arrow keys to change slides
